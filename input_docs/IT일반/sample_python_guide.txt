Python 프로그래밍 실전 가이드

Python은 1991년 귀도 반 로섬이 개발한 고급 프로그래밍 언어이다. 간결하고 읽기 쉬운 문법으로 초보자부터 전문가까지 폭넓게 사용된다.

자료구조
Python의 기본 자료구조는 리스트, 튜플, 딕셔너리, 셋 네 가지이다. 리스트는 가변(mutable)이고 순서가 있으며, 튜플은 불변(immutable)이다. 딕셔너리는 키-값 쌍으로 데이터를 저장하고, 셋은 중복 없는 집합을 표현한다.

리스트 컴프리헨션은 Python의 강력한 기능으로, 반복문과 조건문을 한 줄로 표현할 수 있다. 예를 들어 [x**2 for x in range(10) if x % 2 == 0]은 0부터 9까지 짝수의 제곱 리스트를 생성한다.

함수와 데코레이터
함수는 def 키워드로 정의하며, *args와 **kwargs로 가변 인자를 처리할 수 있다. lambda 표현식으로 익명 함수를 만들 수 있고, map, filter, reduce 함수와 함께 함수형 프로그래밍 패턴을 구현할 수 있다.

데코레이터는 함수를 감싸서 기능을 추가하는 패턴이다. @property, @staticmethod, @classmethod 등이 내장 데코레이터이며, 사용자 정의 데코레이터로 로깅, 캐싱, 인증 등을 구현할 수 있다.

비동기 프로그래밍
asyncio 모듈은 비동기 I/O를 지원한다. async/await 문법으로 코루틴을 정의하고, 이벤트 루프에서 실행한다. 네트워크 요청, 파일 I/O 등 대기 시간이 긴 작업에서 성능을 크게 향상시킬 수 있다.

aiohttp, httpx 등의 라이브러리를 사용하면 비동기 HTTP 클라이언트/서버를 구축할 수 있다.
